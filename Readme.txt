1. Clone the Repo
2. Open the React proj in Visual Studio Code or other IDE.
3. Open the Java spring boot proj in eclipse or other IDE.
4. Copy paste the SQL commands and create the schema and tables in MySQL or other DB.
5. If you are not using MySQL then change the dependencies and properties in java proj accordingly.
6. "npm run dev" copy this command and paste it in the React proj terminal to start the project on port 3000.
7. Run the java spring boot proj.

Good to go!!!..
